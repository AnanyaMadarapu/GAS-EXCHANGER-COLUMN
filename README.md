# CO2 Absorption in Amine-Based Tray Column

## Overview
This project simulates stage-wise absorption of CO2 in an amine-based tray column using simplified equilibrium and operating relationships.

## Objectives
- Model gas-liquid mass transfer across multiple stages
- Analyze CO2 removal efficiency
- Study the effect of number of trays on performance

## Methodology
- Assumed linear equilibrium relation: y = mx
- Implemented stage-wise reduction in gas concentration
- Evaluated system performance across varying stages

## Results
- CO2 concentration decreases progressively across stages
- Higher number of trays leads to increased removal efficiency

## Key Insights
- Tray count significantly impacts absorption efficiency
- Simplified modeling helps understand process trends

## Tools Used
- Python
- NumPy
- Matplotlib
